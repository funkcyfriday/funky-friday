## 🐛 Common Issues and Fixes

### ❌ Script executes but GUI doesn't appear
**Fix:** Press `RightShift` or `Insert` to toggle GUI visibility. If still not visible, disable Discord overlay, GeForce Experience, or any ReShade filters — then re-inject.

### ❌ Autoplay not hitting notes / missing beats
**Fix:** This almost always means the game updated and changed its note detection logic. Check this repo for an updated version — the **funky friday script auto** module is the first thing patched after every Funky Friday update.

### ❌ Animation script not applying animations
**Fix:** The animation unlock is client-side. If animations aren't showing, toggle the module off, wait for the next round to start, then re-enable it. Ensure you're on the latest script version.

### ❌ Auto Farm stops collecting after a few rounds
**Fix:** Toggle Auto Farm off and on to reset the XP detection loop. If the issue persists across sessions, update to the latest version — stage layout changes break the pathing occasionally.

### ❌ Script not working on Android via Delta
**Fix:** Confirm you launched Roblox through Delta's built-in launcher, not independently. Also ensure you're using **Delta v2.3+** — older versions have known incompatibilities with the **funky friday script delta mobile** loader format.

### ❌ "attempt to index nil value" Lua error on execution
**Fix:** The game updated and a variable reference is broken. Check the issues tab or Discord for a hotfix announcement, or wait for the next scheduled patch release.

### ❌ Score Multiplier not reflecting in leaderboard
**Fix:** The multiplier operates client-side for visual feedback during the round. Server-synced leaderboard scores depend on the game's validation — this feature works best in casual lobbies rather than ranked sessions.
